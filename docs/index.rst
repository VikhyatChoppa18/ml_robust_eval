Welcome to ML Eval Robust's documentation!
==========================================

.. image:: https://github.com/VikhyatChoppa18/ml_robust_eval/blob/main/assets/ML%20Eval.png
   :width: 200px
   :align: center

**ML Eval Robust** is a python package, object-oriented library for comprehensive machine learning model evaluation, validation, and robustness testing.

Features:
---------
- Metrics for classification, regression, NLP, and CV tasks
- Cross-validation and A/B testing helpers
- Visualization tools for confusion matrices, ROC curves, etc.
- Automated edge, adversarial, and boundary test case generation
- No external dependencies
